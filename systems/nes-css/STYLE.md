# NES.css

> 8-bit 像素风 CSS 框架，复古任天堂美学

## 设计哲学

用 CSS 还原 8-bit 时代的视觉语言：粗边框、像素字体、硬边缘、无渐变。每个组件都是对 NES 时代 UI 的致敬。

## 关键 Token

| Token | 值 |
|-------|---|
| `$border-size` | `4px` |
| `$color-black` | `#212529` |
| `$color-white` | `#fff` |
| `$default-colors-hover` | `#e7e7e7` |
| `$default-colors-shadow` | `#adafbc` |
| `$disabled-colors-normal` | `#d3d3d3` |
| `$disabled-colors-shadow` | `#adafbc` |
| `$error-colors-hover` | `#ce372b` |
| `$error-colors-normal` | `#e76e55` |
| `$error-colors-shadow` | `#8c2022` |

## ✅ 应该做的

- ✅ 使用 Press Start 2P 像素字体
- ✅ 保持 4px 粗边框的硬边缘风格
- ✅ 用 NES 调色板的饱和色（蓝/绿/黄/红）
- ✅ 组件之间留充足间距

## ❌ 不应该做的

- ❌ 不要用圆角——NES 时代没有圆角
- ❌ 不要用渐变或阴影模糊——硬边缘是核心
- ❌ 不要混入现代字体的正文文字

## 适用场景

- 游戏界面
- 复古主题活动页
- 怀旧风个人博客
- 像素艺术社区
