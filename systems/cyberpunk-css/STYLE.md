# cyberpunk-css

> 赛博朋克 2077 风格 CSS，霓虹黄 + 锯齿字体 + 未来主义界面

## 设计哲学

以《赛博朋克 2077》为灵感的纯 CSS 主题。霓虹黄底 + 黑色文字 + 青色高亮 + 锯齿状未来字体——这不是优雅，这是"数字朋克"的视觉暴力。

## 关键 Token

| Token | 值 |
|-------|---|
| `--banner-color1` | `var(--purple)` |
| `--banner-color2` | `var(--red)` |
| `--banner-color3` | `var(--yellow)` |
| `--banner-color4` | `var(--cyan)` |
| `--bg` | `transparent` |
| `--black` | `#000` |
| `--blue` | `#136377` |
| `--cyan-cyberpunk-font1` | `#46dffb` |
| `--cyan-cyberpunk-font2` | `#45aefb` |
| `--green` | `#446d44` |

## ✅ 应该做的

- ✅ 使用高对比度霓虹色：黄底黑字 + 青色链接
- ✅ 字体选择带有科技感/锯齿感的显示字体
- ✅ 界面元素可以有斜切角和不规则形状
- ✅ 保持信息密度高

## ❌ 不应该做的

- ❌ 不要用柔和的配色
- ❌ 不要用圆角——斜切角更贴合主题
- ❌ 不要大量留白——信息密集才有沉浸感

## 适用场景

- 赛博朋克主题游戏
- 科幻产品发布页
- 未来主义 Dashboard
- 电子音乐活动页
