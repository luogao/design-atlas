# 7.css

> Windows 7 Aero Glass 风格 CSS，毛玻璃 + 渐变的现代桌面美学起点

## 设计哲学

在 XP.css 之上构建 Aero Glass 设计语言。`--w7-*` 命名系统覆盖每个组件的每个状态——从背景到渐变到阴影。这是"操作系统级设计系统"的最完整 CSS 实现。

## 关键 Token

| Token | 值 |
|-------|---|
| `--w7-el-bg` | `#f2f2f2` |
| `--w7-el-bg-d` | `#f4f4f4` |
| `--w7-el-bg-s-1` | `#ebebeb` |
| `--w7-el-bg-s-2` | `#cfcfcf` |
| `--w7-font` | `9pt "Segoe UI", "SegoeUI", "Noto Sans", sans-serif` |
| `--w7-li-bg-hl` | `linear-gradient(#fff9, #e6ecf5cc 90%, #fffc)` |
| `--w7-surface` | `#f0f0f0` |
| `--w7-el-bd` | `#8e8f8f` |
| `--w7-el-bd-a` | `#6d91ab` |
| `--w7-el-bd-d` | `#adb2b5` |

## ✅ 应该做的

- ✅ 使用 Segoe UI 9pt 字体
- ✅ 用 Aero 毛玻璃半透明效果
- ✅ 保持 #f0f0f0 基础表面色
- ✅ 组件状态覆盖完整：normal/hover/active/disabled

## ❌ 不应该做的

- ❌ 不要用扁平化——Aero 的核心是微妙的渐变
- ❌ 不要忽视 scoped 版本
- ❌ 不要用粗边框——Win7 用精致 1px 边框

## 适用场景

- Windows 7 风格应用
- Aero Glass 主题项目
- 桌面软件 Web UI
- 操作系统模拟器
