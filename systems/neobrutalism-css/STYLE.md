# NeoBrutalismCSS

> 新粗野主义 CSS 框架，粗黑边框 + 硬阴影 + 高饱和色块

## 设计哲学

拒绝圆角、拒绝模糊阴影、拒绝"精致"。粗黑边框 + 偏移硬阴影 + 高饱和色块——用原始的视觉冲击力打破千篇一律的"安全设计"。

## ✅ 应该做的

- ✅ 使用粗黑边框（2-4px solid black）
- ✅ 用偏移硬阴影（box-shadow: 4px 4px 0 black）
- ✅ 大胆使用高饱和色块
- ✅ 字体用粗体 sans-serif

## ❌ 不应该做的

- ❌ 不要用圆角——粗野主义的灵魂是直角
- ❌ 不要用模糊阴影或光晕效果
- ❌ 不要用渐变——纯色色块才有力量

## 适用场景

- 创意工作室作品集
- 独立产品 Landing Page
- Z 世代社区平台
- 艺术展览页面
