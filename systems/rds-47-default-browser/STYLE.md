# Default Browser

## 一句话

1994 年的 Mosaic。蓝色下划线链接，灰色背景，这就是 Web。

## 设计哲学

Default Browser 1993 美学是'Web 的原点'——没有任何 CSS（CSS 1996 年才诞生），只有 HTML 标签的默认渲染。灰色背景、Times New Roman 字体、蓝色下划线链接、紫色已访问链接。它的价值在于'回到原点'的纯粹。

## Do / Don't

- ✅ 零 CSS（用 HTML 标签默认样式）
- ✅ Times New Roman / serif 字体
- ✅ 蓝色下划线链接（#0000EE）
- ✅ 灰色背景

- ❌ 不要加任何 CSS
- ❌ 不要用 div 布局（用 table 或纯流式）
- ❌ 不要用无衬线字体
- ❌ 不要用 border-radius

## 适用场景

互联网考古、学术内容、极简博客、Web 起源致敬
