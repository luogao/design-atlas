# Web 2.0 Glossy

## 一句话

beta 标签 + 巨大 glossy 按钮 + AJAX 加载动画。

## 设计哲学

Web 2.0 的视觉语言：glassy/glossy 按钮、大量渐变和倒影、圆角卡片、大号字体的 logo（通常带 beta 标签）、加载动画。设计上追求'Web 应用 = 桌面应用'的感觉。

## Do / Don't

- ✅ 渐变按钮 + 内阴影高光
- ✅ 大圆角卡片容器
- ✅ beta 徽章和标签
- ✅ 反射/倒影效果

- ❌ 不要用扁平设计
- ❌ 不要用暗色主题
- ❌ 不要用 Material Design 的阴影规范
- ❌ 不要省略 glossy 效果

## 适用场景

Web 2.0 怀旧、SaaS 产品致敬、千禧年风格 UI
