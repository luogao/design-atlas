# 分类体系

> 按"视觉来源"分类 — 人类直觉能理解这个东西是从哪来的。

## 8 大类

### 1. OS Interfaces（操作系统界面）

操作系统 GUI 美学。每个时代的操作系统都有独特的视觉语言：bevel/chrome（Win95）、aqua 凝胶感（Mac OS X）、luna 蓝色 glossy（WinXP）。

**共性**：窗口边框、按钮状态（normal/hover/active/disabled）、标题栏、滚动条、对话框。

**来源示例**：Mac System 7, Windows 95, Windows XP Luna, Mac OS X Aqua, NeXTSTEP, BeOS, Amiga Workbench, OS/2 Warp, Mac OS 9 Platinum, TempleOS

### 2. Terminal & TUI（终端美学）

命令行/文本界面美学。从 IBM 大型机到现代 TUI 工具，字符是唯一的视觉元素。

**共性**：等宽字体、有限色板（通常 16 色）、ASCII art、ANSI 转义码、高对比度。

**来源示例**：IBM 3270, CRT Phosphor, DOS CGA, Commodore 64 BASIC, NetHack ASCII, BBS ANSI Art, Midnight Commander, btop Meters

### 3. Gaming（游戏 UI）

游戏机/游戏界面美学。硬件限制催生的独特像素美学，以及游戏 HUD 的信息布局。

**共性**：像素艺术、有限调色板、状态条/HUD、像素字体、dithering 抖动。

**来源示例**：8-Bit Arcade, Game Boy DMG (4色绿), PS1 Tech, NES.css, 98.css, cs16.css

### 4. Design Movements（设计运动）

20 世纪经典设计运动。这些是现代平面设计的根基，有完整的设计哲学。

**共性**：强烈的设计原则（形式追随功能、网格系统等）、理论支撑、可追溯到具体设计师/学校。

**来源示例**：Bauhaus (1919), Swiss Style / 国际主义 (1950s), Braun / Dieter Rams (1960s), Memphis (1980s), Monochrome Zen (2010s)

### 5. Web Eras（Web 时代）

Web 设计的时代演进。每个 web 时代都有技术驱动的视觉特征。

**共性**：浏览器技术限制、CSS 能力的演进、用户习惯的变化。

**来源示例**：Default Browser (1993), GeoCities Web 1.0 (1996), Maximalist 90s Banner (1997), Web 2.0 Glossy (2005), Frutiger Aero (2007), Flat Design (2013), Brutalist Web (2016), Glassmorphism (2020), Neumorphism (2020), Claymorphism (2021)

### 6. Retro-Futurism（复古未来）

复古未来主义/赛博美学。每个时代对"未来"的想象，回头看都有独特的美感。

**共性**：霓虹色、扫描线、故障效果、CRT 弯曲、怀旧感与科技感的矛盾统一。

**来源示例**：Cassette Futurism (1979), Tron Vector (1982), VHS Tracking (1980), Matrix Rain (1999), Y2K Chrome (2000), Glitch Databend (2010), Vaporwave (2011), Hypnagogic (2015)

### 7. Print & Craft（印刷工艺）

印刷工艺和技术绘图美学。这些风格来自物理世界的制图工艺。

**共性**：网点、套色、网格纸、工程图纸、限制色印刷。

**来源示例**：Risograph (1986), Duotone Poster (1960s), Blueprint / CAD (1960s), Grid Paper, Wireframe Sketch (2010)

### 8. Art（艺术风格）

纯艺术运动的 UI 转译。把绘画/视觉艺术的风格语言应用到界面设计中。

**共性**：强烈的视觉冲击、艺术史背景、大胆的图形元素。

**来源示例**：Pop Art (Lichtenstein, 1960s), Op Art (1964)

## 多维度 Tags

除了大类，每个系统可以有多维度标签做交叉筛选：

| 维度 | 标签 |
|------|------|
| **mood** | minimal, bold, playful, dark, warm, futuristic, nostalgic, elegant, raw |
| **palette** | monochrome, duotone, neon, primary-colors, earth-tones, pastel, high-contrast |
| **typography** | serif, sans-serif, monospace, display, handwritten, pixel |
| **density** | dense, spacious, information-rich |
| **best_for** | dashboard, landing-page, portfolio, blog, game, tool, poster |

### 使用示例

"我想找一个 dark + neon + 适合 dashboard 的风格"
→ 从 `mood:dark` + `palette:neon` + `best_for:dashboard` 交叉筛选

"我想做一个温暖人文感的博客"
→ 从 `mood:warm` + `best_for:blog` 筛选
